## 🚀 Getting Started

To run any `.c` file, you’ll need a C compiler like `gcc`.

### 🔧 Compilation Example (Linux/Mac)

```bash
gcc filename.c -o output
./output
```

### 🪟 Compilation Example (Windows - using GCC from MinGW)

```bash
gcc filename.c -o output.exe
output.exe
```
